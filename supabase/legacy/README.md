# Legacy SQL Archive

This directory contains archived SQL files from earlier iterations of the project.

They include:

- pre-`cp_` initialization scripts
- split setup scripts from the early MVP phase
- NextAuth-era auth table SQL
- one-off security and RLS repair scripts

These files are retained for reference only.

Do not use this directory as the default database setup path for the current project.

For the active auth setup, use:

- `../better-auth-tables-cp.sql`
