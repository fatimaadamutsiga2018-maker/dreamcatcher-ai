# ClarityPath 64卦完整解读模板

> 为克总整理的64卦英文解读，用成功学+心理学方式包装，面向欧美市场

---

## 解读风格指南

### 语言风格
- 使用现代心理学和成功学术语
- 避免东方神秘主义词汇
- 强调个人成长、能量管理、决策科学
- 用"能量"、"状态"、"时机"替代"卦象"、"吉凶"

### 结构模板
```
核心主题：一句话概括（用西方概念）
能量状态：描述当前心理/能量状态
行动建议：3条可执行建议（成功学风格）
时机判断：NOW/WAIT/ADJUST
风险提示：1条心理学洞察
```

## 使用说明

### 数据库结构建议
```sql
CREATE TABLE hexagrams_64 (
    id INTEGER PRIMARY KEY,
    sequence_number INTEGER,  -- 传统卦序 1-64
    name_cn TEXT,
    name_en TEXT,
    upper_trigram TEXT,       -- 上卦
    lower_trigram TEXT,       -- 下卦
    core_theme TEXT,          -- 核心主题
    energy_state TEXT,        -- 能量状态描述
    action_1 TEXT,           -- 行动建议1
    action_2 TEXT,           -- 行动建议2
    action_3 TEXT,           -- 行动建议3
    timing_advice TEXT,      -- NOW/WAIT/ADJUST
    risk_warning TEXT,       -- 风险提示
    category TEXT            -- 分类标签
);
```
