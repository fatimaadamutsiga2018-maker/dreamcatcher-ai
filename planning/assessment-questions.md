# ClarityPath 能量测评题库

## 测评说明

- **题目数量：** 12题
- **评分方式：** 每题1-10分
- **测评维度：** 4个维度，每维度3题
- **完成时间：** 约3-5分钟

---

## 维度A：心理状态（Mental Clarity）

### A1. 思绪清晰度
**问题：** 过去一周，你的思绪清晰度如何？

**评分说明：**
- 1分 = 混乱、难以集中注意力、思绪飘散
- 5分 = 时好时坏，需要努力才能保持专注
- 10分 = 思路清晰，专注力强，能深入思考

---

### A2. 情绪稳定性
**问题：** 面对压力或突发状况时，你的情绪稳定度？

**评分说明：**
- 1分 = 容易焦虑、情绪波动大、难以平复
- 5分 = 多数时候能控制，但偶尔还是会失控
- 10分 = 情绪平稳，能快速调整，不被外界轻易影响

---

### A3. 未来期待感
**问题：** 你对未来的整体感受？

**评分说明：**
- 1分 = 充满担忧，看不到希望，觉得未来会更糟
- 5分 = 有期待也有担忧，摇摆不定，说不清楚
- 10分 = 充满期待，相信未来会更好，有清晰愿景

---

## 维度B：身体状态（Physical Vitality）

### B4. 晨间精力
**问题：** 早晨醒来时的精力水平？

**评分说明：**
- 1分 = 疲惫，需要很长时间才能清醒，想继续睡
- 5分 = 一般，需要咖啡或闹钟才能起床，勉强开始一天
- 10分 = 自然醒，精神饱满，身体准备好迎接新的一天

---

### B5. 睡眠质量
**问题：** 过去一周的睡眠质量？

**评分说明：**
- 1分 = 失眠、多梦、易醒，睡后仍感疲惫
- 5分 = 有时好有时差，不够稳定，勉强够用
- 10分 = 入睡快、睡得深、醒来感觉 refreshed 和恢复

---

### B6. 身体活力
**问题：** 整体身体的轻盈感和活力？

**评分说明：**
- 1分 = 沉重、疲惫、经常不适，身体像是负担
- 5分 = 一般，没有大问题但也不轻盈，中规中矩
- 10分 = 轻盈、有活力、身体配合度高，感觉充满能量

---

## 维度C：生活满意度（Life Harmony）

### C7. 事业投入感
**问题：** 对当前工作/事业的投入和满足感？

**评分说明：**
- 1分 = 厌倦、想逃离、没有价值感，只是混日子
- 5分 = 还行，有稳定收入但缺乏激情，不上不下
- 10分 = 热爱、有使命感、充满动力，愿意投入额外精力

---

### C8. 关系质量
**问题：** 重要关系（家庭/伴侣/挚友）的质量？

**评分说明：**
- 1分 = 紧张、冲突、感到孤独，关系是消耗
- 5分 = 平淡，维持但缺乏深度连接，可有可无
- 10分 = 滋养、支持、深度连接，关系是能量来源

---

### C9. 财务安全感
**问题：** 当前的财务安全感？

**评分说明：**
- 1分 = 焦虑、入不敷出、没有保障，财务是巨大压力
- 5分 = 基本够用，但没有余裕，不敢有大支出
- 10分 = 稳定、有储蓄、对未来有信心，财务是支撑

---

## 维度D：成长动力（Growth Momentum）

### D10. 学习热情
**问题：** 学习新事物或技能的兴趣？

**评分说明：**
- 1分 = 没有动力，觉得太累，不想学新东西
- 5分 = 有兴趣但缺乏行动，知道该学但没开始
- 10分 = 充满好奇，主动探索新知识，享受学习过程

---

### D11. 自我投资
**问题：** 在自我提升上的时间和金钱投入？

**评分说明：**
- 1分 = 几乎没有，顾不上自己，总是先满足别人
- 5分 = 偶尔，但不系统，想起来才做
- 10分 = 持续投入，把成长当优先事项，有固定安排

---

### D12. 挑战态度
**问题：** 面对挑战和变化时的态度？

**评分说明：**
- 1分 = 回避、害怕、觉得无力应对，希望一切不变
- 5分 = 视情况而定，有时能面对，有时想逃避
- 10分 = 迎接挑战，相信是成长机会，主动寻求突破

---

## 评分计算

### 总分计算
```
总分 = (A1 + A2 + A3 + B4 + B5 + B6 + C7 + C8 + C9 + D10 + D11 + D12) / 12
```

### 维度分计算
```
心理维度分 = (A1 + A2 + A3) / 3
身体维度分 = (B4 + B5 + B6) / 3
生活维度分 = (C7 + C8 + C9) / 3
成长维度分 = (D10 + D11 + D12) / 3
```

### 能量类型判定

| 总分区间 | 能量类型 | 英文名称 | 特征描述 |
|---------|---------|---------|---------|
| 8.5 - 10.0 | 高能状态 | High Flow | 能量充沛，适合推进重要事项 |
| 6.5 - 8.4 | 平衡状态 | Steady State | 能量平稳，适合巩固和优化 |
| 4.5 - 6.4 | 调整期 | Recalibration | 能量波动，需要重新校准方向 |
| 1.0 - 4.4 | 充电期 | Recharge Mode | 能量低落，优先休息和恢复 |
