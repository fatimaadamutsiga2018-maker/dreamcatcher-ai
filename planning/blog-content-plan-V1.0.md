# ClarityPath 博客内容规划

## 博客定位

**目的：** SEO引流 + 品牌认知 + 软广转化  
**风格：** 温暖、有洞察力、实用、不鸡汤  
**目标读者：** 35+寻求个人成长和决策指引的人群

---

## 内容分类

### 1. 能量管理（Energy Management）
帮助读者理解和管理自己的能量状态

**选题方向：**
- 如何识别自己的能量低谷信号
- 高能人士的5个日常习惯
- 为什么你越休息越累？能量管理的真相
- 从"时间管理"到"能量管理"的转变

### 2. 决策智慧（Decision Wisdom）
在不确定中做出更好选择的方法

**选题方向：**
- 面对重大选择时，如何倾听内心的声音
- 为什么直觉有时候比分析更准
- 决策疲劳：为什么小事消耗你的大事判断力
- 古老智慧如何帮助现代决策

### 3. 个人成长（Personal Growth）
35+人群的转型和成长

**选题方向：**
- 35岁不是终点，而是新的起点
- 如何在变动中找到自己的锚点
- 重建内心秩序：从混乱到清晰的旅程
- 中年转型的3个关键步骤

### 4. 东方哲学（Eastern Philosophy）
东方智慧的现代解读

**选题方向：**
- 顺势而为：不是放弃，而是智慧的选择
- 道家"无为"在现代生活中的应用
- 易经智慧：变化是唯一的不变
- 东方哲学中的"平衡"概念

## 文章结构模板

### 标准文章结构

```
1. 引人入胜的开头（Hook）
   - 一个场景、故事或问题
   - 与读者痛点共鸣

2. 问题阐述
   - 为什么这个问题重要
   - 常见误区

3. 核心内容（3-5个点）
   - 每个点有理论+实践
   - 使用小标题、列表、引用

4. 工具/方法介绍（软广插入点）
   - 自然过渡到ClarityPath的功能
   - "你可以用我们的能量测评来了解..."

5. 行动号召（CTA）
   - 开始测评
   - 查看今日能量环境
   - 订阅Newsletter

6. 结尾
   - 总结核心观点
   - 温暖、鼓励的语气
```

## SEO策略

### 关键词布局
- 主关键词：energy assessment, personal growth, decision making
- 长尾关键词：how to manage energy, midlife career change, ancient wisdom modern life

### 技术SEO
- 每篇文章独立URL：/blog/article-title
- 优化meta title和description
- 使用结构化数据（Schema.org）
- 内部链接（文章间互相链接）

### 内容SEO
- 标题包含关键词
- 首段出现关键词
- 使用H2、H3标签
- 图片添加alt文本
